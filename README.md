## Installing

### PECL

	pecl install psi

### PHARext

Watch out for [PECL replicates](https://replicator.pharext.org?psi)
and pharext packages attached to [releases](./releases).

### Checkout

	git clone github.com:m6w6/psi
	cd psi
	/path/to/phpize
	./configure --with-php-config=/path/to/php-config
	make
	sudo make install
